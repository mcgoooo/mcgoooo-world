# mcgoooo-world

personal website of Andrew Hilton

# Getting Started

```
npm install
npm run dev
```

to preview in cloudflare [wrangler](https://developers.cloudflare.com/workers/wrangler/) run the following command
```
npm run preview
```

this is part of cloudflares efforts towards [Opennext](https://opennext.js.org/) goals and to have a reproducible environment
